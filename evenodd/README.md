# Even-Odd Program
This is Day 2 of my C Basics Daily Learning journey.
Description:
This program checks whether a given number is even or odd. It demonstrates the use of conditional statements to make decisions based on user input.

Features:
- Accepts a number from the user
- Determines if it is even or odd
- Displays the result

How it works:
1. User enters a number.
2. Program checks if number % 2 == 0:
   - If yes → number is even
   - If no  → number is odd
3. Result is displayed.

Example Output :
```
Enter a value for num : 3
Given Number is Odd

Enter a Value for num : 6
Given Number is Even
```

------------

2. Compile if required 
3. Run the program:
