# Welcome Program

This C program prints a welcome message for the **C Basics Daily Learning**.

## Concepts used
- Basic C syntax
- `printf` function

## Example Output
Welcome to C Basics Daily Learning !!

## How to Run
1. Open any C compiler (like Turbo C, Code::Blocks, Dev C++, or VS Code).  
2. Compile `welcome.c`.  
3. Run the executable to see the output.

**Note for Turbo C users:**  
Use `#include <conio.h>`, `void main()`, and `getch()` instead of `int main()` and `return 0`.
