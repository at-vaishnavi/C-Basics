# Simple Calculator
This is **Day 1** of my C Basics Daily Learning journey.

This C program implements a **simple calculator** using **conditional statements**.  
It can perform **addition, subtraction, multiplication, and division** based on the operator entered by the user.

## Concepts Used
- Basic C syntax
- `printf` and `scanf` functions
- `if` / `else if` / `else` conditional statements
- Arithmetic operators: `+`, `-`, `*`, `/`

## Example Output
```

Enter a valid operator (+, -, *, /): +
Enter values for a and b: 5 3
Addition of a and b: 8

Enter a valid operator (+, -, *, /): /
Enter values for a and b: 10 0
Zero Division Error !
```


## How to Run
1. Open any C compiler (like Turbo C, Code::Blocks, Dev C++, or VS Code).  
2. Compile `calci.c`.  
3. Run the executable and follow the prompts to enter operator and values.
