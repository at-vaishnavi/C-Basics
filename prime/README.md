# Prime Number Program
This is Day 2 of my C Basics Daily Learning journey.

Description:
This program checks whether a given number is a prime number. A prime number is a number greater than 1 that has no divisors other than 1 and itself.

Features:
- Accepts input from the user
- Determines if the number is prime or not
- Displays the result clearly

How it works:
1. User enters a number.
2. Program checks divisibility of the number from 2 up to the square root of the number.
   - If divisible → number is not prime
   - If not divisible → number is prime
3. Displays the result.

Example Output
```
Enter a number: 10
10 is not a prime number

Enter a number: 7
7 is a prime number
```
How to Run:
------------
1. Save the program code in a file 
2. Compile if required 
3. Run the program:
4. Enter the number when prompted.
