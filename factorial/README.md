# Factorial Program (Using Loop)
This is **Day 3** of my C Basics Daily Learning journey.

This C program calculates the **factorial of a number** by taking input from the user.  
It uses a **for loop** to compute the factorial.

## Concepts Used
- Basic C syntax
- `printf` and `scanf` functions
- `for` loop
- Variables and data types

## Sample Output

## How to Run
1. Open any C compiler (like Turbo C, Code::Blocks, Dev C++, or VS Code).  
2. Compile `fact.c`.  
3. Run the executable and enter a number when prompted.
